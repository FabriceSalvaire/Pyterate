3. Installation¶

Note to Packagers: Please don’t create AutoSphinx package (PiPY do the job)

3.1. On Windows¶

Firstly, you have to install the Anaconda Distribution so as to get a full featured Python 3 environment.

Then open the Anaconda Navigator and launch a console for your root environment.

You can now run pip to install AutoSphinx in your root environment using this command:

pip install AutoSphinx

3.2. On Linux¶

Firstly, you have to install Python 3 from your distribution.

Then you can install AutoSphinx using pip or from source. See supra.

3.3. On OSX¶

There are several ways to get Python on OSX:

  • use the built in Python
  • install Miniconda
  • install the Anaconda Distribution.
  • install from Brew brew install python3 (reported to work)

You can install AutoSphinx using pip or from source. See supra.

3.4. Installation from PyPi Repository¶

AutoSphinx is available on the Python Packages PyPI repository at https://pypi.python.org/pypi/AutoSphinx

Run this command in the console to install the latest release:

pip install AutoSphinx

3.5. Install a more recent version from Github¶

If you want to install a version which is not yet released on Pypi, you can use one of theses commands to install the stable or devel branch:

pip install git+https://github.com/FabriceSalvaire/AutoSphinx

pip install git+https://github.com/FabriceSalvaire/AutoSphinx@devel

3.6. Installation from Source¶

The AutoSphinx source code is hosted at https://github.com/FabriceSalvaire/AutoSphinx

You have to solution to get the source code, the first one is to clone the repository, but if you are not familiar with Git then you can simply download an archive either from the AutoSphinx Pypi page (at the bottom) or the GitHub page (see clone or download button).

To clone the Git repository, run this command in a console:

git clone https://github.com/FabriceSalvaire/AutoSphinx.git

Then to build and install AutoSphinx run these commands:

python setup.py build
python setup.py install

3.7. Dependencies¶

AutoSphinx requires the following dependencies:

  • Python 3
  • PyYAML

To generate the documentation, you will need in addition:

  • Sphinx
